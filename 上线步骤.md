# PicTool 上线步骤（手把手教程）

## 方案一：免费上线（推荐新手）

### 使用 GitHub Pages（完全免费）

**步骤1：注册GitHub账号**
1. 打开 https://github.com
2. 点击 "Sign up"
3. 填写邮箱、密码、用户名
4. 验证邮箱

**步骤2：创建仓库**
1. 登录GitHub
2. 点击右上角 "+" → "New repository"
3. 仓库名填：pictool（或你喜欢的名字）
4. 选择 "Public"
5. 点击 "Create repository"

**步骤3：上传文件**
1. 在新仓库页面，点击 "uploading an existing file"
2. 选择 pic-tool.html 文件
3. 点击 "Commit changes"

**步骤4：开启GitHub Pages**
1. 仓库页面点击 "Settings"
2. 左侧菜单点击 "Pages"
3. Source 选择 "Deploy from a branch"
4. Branch 选择 "main" → "/ (root)"
5. 点击 "Save"
6. 等待几分钟，会显示网站链接：https://你的用户名.github.io/pictool/

**完成！你的网站上线了。**

## 方案二：国内访问更快（推荐）

### 使用 Gitee Pages（国内版GitHub）

**步骤1：注册Gitee账号**
1. 打开 https://gitee.com
2. 点击注册，用手机号注册

**步骤2：创建仓库**
1. 点击右上角 "+" → "新建仓库"
2. 仓库名：pictool
3. 选择 "公开"
4. 点击 "创建"

**步骤3：上传文件**
1. 点击 "上传文件"
2. 选择 pic-tool.html
3. 点击 "提交"

**步骤4：开启Gitee Pages**
1. 仓库页面点击 "服务" → "Gitee Pages"
2. 选择 "部署分支"：master
3. 点击 "启动"
4. 等待审核（约1天）
5. 获得链接：https://你的用户名.gitee.io/pictool/

## 方案三：专业级（有域名）

### 使用 Vercel（免费，性能更好）

**步骤1：准备**
1. 注册GitHub账号（同上）
2. 上传代码到GitHub仓库

**步骤2：注册Vercel**
1. 打开 https://vercel.com
2. 点击 "Sign Up" → 用GitHub登录
3. 授权Vercel访问你的GitHub

**步骤3：部署**
1. 点击 "Add New Project"
2. 选择你的 pictool 仓库
3. 点击 "Import"
4. Framework Preset 选择 "Other"
5. 点击 "Deploy"
6. 等待1分钟，自动获得域名：https://pictool-随机字符.vercel.app

**步骤4：绑定自己的域名（可选）**
1. 购买域名（阿里云/腾讯云）
2. Vercel项目设置 → Domains
3. 添加你的域名
4. 按提示配置DNS

## 申请 Google AdSense（变现关键）

### 前提条件
- 网站已上线
- 有原创内容（工具页面算内容）
- 网站能正常访问

### 申请步骤

**步骤1：注册**
1. 打开 https://www.google.com/adsense
2. 点击 "开始使用"
3. 登录Google账号
4. 填写网站URL
5. 填写个人信息（姓名、地址、电话）

**步骤2：等待审核**
- 审核时间：1天到2周不等
- 期间继续优化网站内容

**步骤3：通过审核后**
1. 获取广告代码
2. 将代码添加到网页的 `<head>` 标签内
3. 广告会自动显示在预留的广告位

## 遇到问题怎么办？

1. **网站打不开**
   - 检查文件是否上传成功
   - 检查仓库是否公开
   - 等待10分钟再试（部署需要时间）

2. **AdSense申请被拒**
   - 增加网站内容（写几篇博客文章）
   - 确保网站能正常访问
   - 1个月后重新申请

3. **访问速度慢**
   - 国内用户用Gitee
   - 国外用户用GitHub/Vercel
   - 购买CDN加速（后期）
