## 方案一：GitHub Pages（推荐）

### 优点
- 完全免费
- 自动分配域名：`https://你的用户名.github.io/项目名`
- 稳定可靠
- 支持自定义域名（后期可买）

### 部署步骤（跟着做）

#### 第1步：注册 GitHub 账号（5分钟）

1. 打开 https://github.com
2. 点击绿色按钮 **"Sign up"**
3. 填写信息：
   - Email：你的邮箱
   - Password：密码（记好）
   - Username：用户名（会成为网址的一部分，想好再填）
4. 验证邮箱（去邮箱点链接）

#### 第2步：创建项目（3分钟）

1. 登录 GitHub
2. 点击右上角 **+** 号 → **"New repository"**
3. 填写：
   - Repository name：`pictool`（或你喜欢的名字）
   - Description：`免费在线图片处理工具`（可选）
   - 选择 **Public**（公开）
   - ✅ 勾选 **"Add a README file"**
4. 点击绿色按钮 **"Create repository"**

#### 第3步：上传网站文件（5分钟）

**方法一：网页上传（简单）**

1. 进入你的项目页面
2. 点击 **"Add file"** → **"Upload files"**
3. 把 `pictool-v2.html` 文件拖进去
4. 点击 **"Commit changes"**

**方法二：改名 index.html（必须做）**

1. 点击上传的文件 `pictool-v2.html`
2. 点击右上角的 **铅笔图标**（Edit）
3. 把文件名改成 `index.html`
4. 点击 **"Commit changes"**

#### 第4步：开启网站（2分钟）

1. 项目页面点击 **"Settings"**（顶部标签）
2. 左侧菜单点击 **"Pages"**
3. Source 选择 **"Deploy from a branch"**
4. Branch 选择 **"main"** → 文件夹选 **"/(root)"**
5. 点击 **"Save"**
6. 等待 1-2 分钟

#### 第5步：访问你的网站

刷新页面，会看到一个绿色框：
```
Your site is live at https://你的用户名.github.io/pictool/
```

**点击这个链接，就是你的网站！**

把这个链接发给任何人，他们都能访问。

## 方案二：Gitee Pages（国内访问快）

### 优点
- 国内访问速度快
- 完全免费
- 中文界面

### 部署步骤

#### 第1步：注册 Gitee
1. 打开 https://gitee.com
2. 用手机号注册

#### 第2步：创建仓库
1. 点击右上角 **+** → **"新建仓库"**
2. 仓库名称：`pictool`
3. 选择 **"公开"**
4. 点击 **"创建"**

#### 第3步：上传文件
1. 点击 **"上传文件"**
2. 选择 `pictool-v2.html`
3. 改名为 `index.html`
4. 点击 **"提交"**

#### 第4步：开启 Pages
1. 点击 **"服务"** → **"Gitee Pages"**
2. 选择分支 **"master"**
3. 点击 **"启动"**
4. 等待审核（通常几分钟到几小时）

#### 第5步：访问
网址格式：`https://你的用户名.gitee.io/pictool`

## 常见问题

### Q：需要买域名吗？
**不需要！** GitHub/Gitee 会送你免费域名。

### Q：别人能访问吗？
**能！** 只要你的仓库是 Public（公开），任何人都能访问。

### Q：能绑定自己的域名吗？
**可以，但不是必须的。** 等你赚钱了再买域名也不迟。

### Q：网站会失效吗？
- GitHub Pages：只要 GitHub 不倒，网站一直在
- Gitee Pages：需要定期登录账号保持活跃

### Q：上传文件大小有限制吗？
- 单个文件最大 100MB
- 整个项目建议不超过 1GB

### Q：可以改网站内容吗？
**可以！** 修改 `index.html` 文件，网站会自动更新。
