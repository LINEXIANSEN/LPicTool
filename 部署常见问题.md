# 部署常见问题解答

## 关于"不买服务器、不买域名"

### 问题1：我不想买服务器，能让别人访问我的网站吗？

**答：可以！**

GitHub Pages / Gitee Pages / Vercel 这些公司提供免费服务器：
- 你把代码上传到他们平台
- 他们帮你托管，分配域名
- 完全免费

**类比：**
- 买服务器 = 自己买地皮盖房子
- 用 GitHub Pages = 住免费公寓（房东提供房子）

### 问题2：免费的东西靠谱吗？

**答：非常靠谱！**

- GitHub 是全球最大代码托管平台（微软旗下）
- 无数大公司、开源项目都用它
- 免费版足够个人使用

### 问题3：免费域名是什么样的？

```
GitHub 给你的域名：
https://你的用户名.github.io/pictool

举例：
https://zhangsan.github.io/pictool
https://xiaoming.github.io/pictool
```

**特点：**
- 自动生成，不用买
- 全球可访问
- 支持 HTTPS（安全）

### 问题4：以后可以换成自己的域名吗？

**答：可以！**

步骤：
1. 去阿里云/腾讯云买个域名（60元/年）
2. 在 GitHub Settings 里绑定
3. 用户访问你的域名，自动跳转到网站

**建议：** 先用免费域名，等赚钱了再买。

## 部署失败排查

### 问题：网站显示 404

**原因和解决：**

1. **文件名不对**
   - 必须是 `index.html`
   - 不能是 `pictool-v2.html`
   - 解决方法：重命名文件

2. **Pages 没开启**
   - 去 Settings → Pages 检查
   - 确保选择了 "main" 分支

3. **仓库是 Private（私有）**
   - 免费版 GitHub Pages 必须是 Public
   - 解决方法：Settings → Change visibility → Public

### 问题：网站打开很慢

**原因：** GitHub 服务器在国外

**解决：**
- 国内用户多 → 用 Gitee Pages
- 或者等 1-2 分钟再刷新

### 问题：修改了代码，网站没变化

**原因：** 有缓存

**解决：**
1. 强制刷新：Ctrl + F5（Windows）或 Cmd + Shift + R（Mac）
2. 等 1-2 分钟再试
3. 在网址后面加 `?v=2` 强制刷新

## 关于赚钱

### 问题：网站上线后怎么赚钱？

**答：接入广告**

步骤：
1. 申请 Google AdSense（免费）
2. 获得广告代码
3. 把代码放到网页里
4. 有人访问 → 看到广告 → 你赚钱

**收入预估：**
- 1000 次访问 ≈ 1-5 美元
- 1 万次访问 ≈ 10-50 美元
- 10 万次访问 ≈ 100-500 美元

### 问题：多久能赚钱？

**答：取决于流量**

- 网站上线 ≠ 自动来钱
- 需要推广，让人知道你的网站
- 通常 1-3 个月开始有收入

## 关于维护

### 问题：网站需要维护吗？

**答：基本不需要**

- 代码放上去就不用管
- 用户访问的是静态网页
- 没有数据库，不会崩溃

### 问题：可以改内容吗？

**答：随时可以**

1. 打开 GitHub 项目
2. 编辑 `index.html`
3. 保存后自动更新
